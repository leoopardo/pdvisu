# Input

Input component with different props.

#### Example

<Canvas of={Input.Default} />

## Usage

```ts
import {Input} from "sld-ui";

const Example = () => {
  return (
      <Input
       placeholder="text" 
       disabled={false}
      />
  );
};

export default Example;
```

#### Arguments

- **placeholder** A string that represents the input placeholder.
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
